You are a senior full-stack developer. Build a modern, highly animated business portfolio web application using the following requirements.

## TECH STACK
- Frontend: Next.js (latest, App Router)
- Backend: Node.js with Express
- Database: PostgreSQL
- ORM: Prisma
- Styling: Tailwind CSS
- Animations: Framer Motion

## PROJECT STRUCTURE
Create everything inside ONE ROOT FOLDER with this structure:

/apps/frontend (Next.js app)
/apps/backend (Express server)
/packages/ui (shared UI components)
/db (database schema and config)

Ensure everything runs together with a single command using concurrently.

---

## DESIGN REQUIREMENTS
- Modern, clean, professional UI
- Smooth animations and transitions across the entire site
- Use Framer Motion for:
  - Page transitions
  - Scroll animations
  - Hover effects
- Use glassmorphism and soft shadows
- Responsive on mobile and desktop
- Use a dark theme with accent colors
